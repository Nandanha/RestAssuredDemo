# RestAssuredDemo
RestAssured projet
1.This project has the basic framework for you to work with REST APIs.
2.All the basic test for actions like GET PUT POST DELETE options with various header types are available.
3.Also it has methods to handle token level authentication.
4.Approvals is something new which can handle assertion of complete response.Refer the pom file for approvals and also workflow4.java file 
for its implementation for tests.
5.We have allure reporting which generates very simple presentable report.
